# re-typed

BaseKit for React, Redux, TypeScript + bundler (webpack)
    
## Content

- [x] webpack
- [] tslint
- [x] typescript
- [x] babel
- [x] react
- [x] redux
- [] test

## Setup

```
$ npm install
```

## Run

```
$ npm run dev
```
To serve index.html and compiled output run:
```
$ python -m SimpleHTTPServer 3000
```
